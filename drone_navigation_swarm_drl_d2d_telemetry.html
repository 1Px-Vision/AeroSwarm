LiDAR rays are simulated with Three.js raycasting. SLAM is an occupancy-grid estimate built from LiDAR hits and free-space updates. Depth estimation is a forward low-resolution range image reconstructed from simulated rays.

Deep RL Navigation

DQN-like action values
The DRL mode uses a lightweight DQN-style policy approximation in JavaScript. It reads LiDAR clearance, waypoint bearing, distance, and formation error, then selects actions such as KEEP, SLOW, LEFT, RIGHT, CLIMB, or SAFE.
